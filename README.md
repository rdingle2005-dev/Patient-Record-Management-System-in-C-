# Patient Appointment Management System (C++)

A console-based patient management system built in C++ that demonstrates object-oriented programming, custom data structures, and algorithm implementation.

---

##  Features

- Add, update, delete, and search patient records
- Custom singly linked list implementation
- Insertion sort (alphabetical by name)
- Binary search for efficient patient ID lookup
- File saving/loading (persistent storage)
- Appointment conflict detection
- Input validation and robust date validation (MM-DD-YYYY)

---

## Concepts Demonstrated

- Object-Oriented Programming (Encapsulation, Inheritance, Polymorphism)
- Data Structures (Linked List)
- Algorithms:
  - Insertion Sort
  - Binary Search (via indexed vector)
- File I/O
- Input Validation & Error Handling

---

## Technologies Used
- Dev C++
- C++
- Standard Template Library (vector, string, iostream, fstream)

---

##  Project Structure

- src/source files
- include/header files
- data/saved patient records
